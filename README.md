# ReadingDiary
Приложение читательский дневник, которое позволяет искать книги через OpenLibrary, сохранять их в локальную библиотеку, управлять заметками о книгах и отслеживать прогресс чтения

## Стек технологий:
- **UIKit**
- **Viper + Coordinator**
- **GCD**
- **URLSession** 
- **CoreData**
